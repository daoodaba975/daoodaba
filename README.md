# Daooda

👨🏽‍💻 My personal website built with Next JS 14.

## Getting Started

First, install and run the development server:

```bash
# install dependencies
npm install
# run project
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.
